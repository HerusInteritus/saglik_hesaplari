# Sağlık Hesapları PWA

## GitHub Pages'e yükleme
1. GitHub'da yeni bir repo aç.
2. Bu klasördeki tüm dosyaları repo içine yükle.
3. Repo ayarlarında **Pages** bölümünü aç.
4. Source olarak ana branch'i seç.
5. Site birkaç dakika içinde yayınlanır.

## İçerik
- Erkek yağ oranı
- Kadın yağ oranı
- BMR / TDEE
- Hedef kalori
- BMI
- Bel/Boy oranı
- Yağ kütlesi / yağsız kütle
- FFMI / normalize FFMI
- Protein, yağ ve su önerileri

## Not
Veriler tarayıcıdaki localStorage içinde tutulur. Sunucu tarafında veri saklama yoktur.
